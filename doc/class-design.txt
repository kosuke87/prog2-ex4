class Move(現在位置をもらう){
    p(ユーザー)を移動させる
    s(初期位置)から右に移動させるなら左の数字を+1
                左に移動させるなら-1
                下なら右の数字を+1
                上なら-1
    G(ゴール)にたどり着いたら終了
    #(壁)にいけないようにする

    return 動いた座標
}
